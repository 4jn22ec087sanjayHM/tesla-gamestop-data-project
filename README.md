## 📊 Project Overview

This project analyzes the historical stock prices and revenue of Tesla and GameStop using Python. The data is collected using the yfinance library and web scraping techniques. The project visualizes trends to understand the relationship between stock price movements and company revenue.

## 🛠️ Technologies Used

* Python
* Pandas
* Matplotlib
* yfinance
* BeautifulSoup
* Requests

## 📈 Key Features

* استخراج بيانات الأسهم باستخدام yfinance
* جمع بيانات الإيرادات باستخدام web scraping
* تنظيف البيانات وتحليلها
* إنشاء رسوم بيانية لعرض الاتجاهات

## 📷 Visualizations

* Tesla stock price vs revenue
* GameStop stock price vs revenue

## 🎯 Conclusion

The project demonstrates how stock prices may not always align with company revenue, especially in cases like GameStop where market behavior significantly influenced stock trends.

## 📁 Project Structure

* `historical_Data.py` → main analysis script
* `README.md` → project documentation

## 🚀 How to Run

1. Install required libraries:

```
pip install yfinance pandas matplotlib beautifulsoup4 requests html5lib
```

2. Run the script:

```
python historical_Data.py
```
